### What is Charts?

- It is a `lightweight` chart library based on **canvas**.

### Provides

* `line`
* `bar`
* `pie`
* `radar`
* `gauge`

### Install with npm

```shell
$ npm install @jiaminghi/charts
```

Detailed documents and examples can be viewed on the [HomePage](http://charts.jiaminghi.com).
